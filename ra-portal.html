Assessment Steps:

  1. Explain study / obtain consent
  2. Demonstrate Think-Aloud
  3. Calibrate eye tracking
    (before each game)
  4. Randomize puzzle games
  5. Download data
    (after each game)
  6. Input participant ID#
  7. Upload data
  8. Update participation tracker

Language Guidelines:

  • Model think-then-act language
  • AVOID PRIMING WORDS
    (hypothesis, establish, confirm)
  • Diction is okay to repeat only if first used by participant
  • Speech must be consistent with puzzle movements
    (for silent participants, say "hands off the keyboard")
  • If solution is revealed verbally, immediately end assessment
    (data is now compromised)
